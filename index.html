About

-> Greetings! I'm Tahira Malik, a Frontend Software engineer based in Abbottabad, Pakistan.
I enjoy creating things that live on the internet, whether that be websites, Mobile applications, or anything in between. My goal is to always build products that provide pixel-perfect, performant experiences.
My passion for programming started early at age 15 where I started learning the concepts of basic programming practically, after a year I got introduced with C/C++ language where I started learning the basic concepts of writing basic programs but the more was learning the more curious I got which took me in the depth of programming and started bachelors of computer science, which opened new doors of learning and knowledge for me, then I got familiar to Java, concepts of OOP, database and many more.
The journey was going great but then I found the Tungstun Software House,There I met Arsal and Shah Hassan who was a programming guru for me. They showed me the door of different programming language and it was the time when my brain started pushing notifications. They introduced me to the frontend technologies like HTML, CSS, JavaScript and React etc.
